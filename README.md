# 📊 Análisis de Datos: Superstore Sales 2026
**Repositorio para entregas de Trabajos Prácticos (TP)**

Este proyecto consiste en un análisis exploratorio de datos (EDA) sobre el dataset de una "Superstore". El objetivo es identificar patrones de ventas, categorías más rentables y áreas de oportunidad para el negocio.

## 🚀 Contenido del Repositorio
* `Análisis_Superstore.ipynb`: Cuaderno de Python con todo el código y las visualizaciones.
* `superstore.csv`: Dataset original utilizado para el estudio.
* `README.md`: Descripción del proyecto (este archivo).

## 🛠️ Herramientas Utilizadas
* **Python 3.x**
* **Pandas & NumPy**: Manipulación y limpieza de datos.
* **Matplotlib & Seaborn**: Creación de gráficos estadísticos con estilo personalizado.
* **Google Colab**: Entorno de desarrollo.
